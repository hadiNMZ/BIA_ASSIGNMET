# SVU Store Frontend

Dependency-free frontend for the BIA e-commerce API.

## Run

Open `index.html` in a browser, or serve the folder with any static server.

The app defaults to the live API:

```text
https://api.svu.store.abdulha.de
```

Use the API base URL field on the sign-in screen if your backend runs elsewhere.

## Screens

- Sign in with user ID only. The frontend sends the backend's required placeholder password automatically.
- Home page with recommended products and category tabs.
- Product page with backend-backed ratings and purchase behavior tracking.
